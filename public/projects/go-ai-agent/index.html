Go AI Agent Framework

April 1, 2026 · 1 min · daidaiwoma

项目简介#

用 Go 实现的 AI Agent 基础框架，目标是在生产环境中可靠地运行 LLM 驱动的自动化流程。

核心功能#

  • Tool Calling：结构化工具注册与调用，自动生成 JSON Schema
  • 多轮对话：带上下文窗口管理的对话历史
  • 流式输出：SSE 流式响应，低延迟感知
  • 可观测性：内置 trace、日志和指标埋点

技术栈#

  • 语言：Go 1.22+
  • LLM 接入：OpenAI-compatible API
  • 部署：容器化，支持水平扩展

状态#

🚧 开发中
